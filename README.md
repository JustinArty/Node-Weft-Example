# Node-Weft-Example
A demo project showcasing for [Node-Weft](https://github.com/JustinArty/Node-Weft) library.
## Getting Started.
```
git clone https://github.com/JustinArty/Node-Weft-Example
cmake -B build
cmake --build build
```
## Example1 - 2D lens simulation
This example demonstrates how to integrate the developer’s library—specifically RayOptics.h—into the node system.
